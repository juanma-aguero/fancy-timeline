How to

  • First, include in your html the javascript and css links to the file provided:

    <script type="text/javascript" src="js/fancy-timeline.js"></script>
<link type="text/css" href="css/ftl-default/fancy-timeline.css" rel="stylesheet" />

  • Then you need to generate some data like this:

    var dates = [
  {
  "marker": "marker1", 
  "type": "default",
  "title": "title 1" ,
  "content": "Here is the content 1"
  },
  {
  "marker": "marker2", 
  "type": "default",
  "title": "title 2" ,
  "content": "Here is the content 2"
  }
];

  • Then, select with jquery:

    $(".timeline-container").fancyTimeline({
  initPos: 0,
    data: dates
});
